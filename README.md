# WTWR (What to Wear?): Back End

The back-end is focused on creating a server for the WTWR application. Back end code controls error handling, user creation and athentication/ authorization, and database. 

## Running the Project

`npm run start` — to launch the server

`npm run dev` — to launch the server with the hot reload feature

## Features

- Express routing to communicate with MongoDB
- Uses Mongoose Schemas
- Custom error handling
- User creation and editing with name, avatar, email, and password encryption
- Authorization and Authentication using JWT
- Deployed to Google Cloud (no longer live)
- Back end deployed to sub-domain (no longer live)
